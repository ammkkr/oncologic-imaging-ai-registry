# Detailed supplementary files

- `Multimedia_Appendix_1_Detailed_Audit.pdf` contains Supplementary Methods and Tables S1-S21. Table S17 lists all 263 NCT identifiers in the strict primary cohort; Tables S18-S21 provide record-level audits of design reclassification, downstream endpoints, and completed-study result reporting.
- `Multimedia_Appendix_2_Record_Level_Data.xlsx` contains filterable sheets for all 846 candidate records, the 263-study strict primary cohort, each prespecified audit subset, the codebook, and the exact search queries.

The files contain public study-level registry metadata only. They include no participant-level data, protected health information, credentials, local file paths, or medical images.
